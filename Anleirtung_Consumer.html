AddHashtag: Dateiorganisation neu gedacht

Warum AddHashtag?

Mit AddHashtag lösen Sie sich von starren, unübersichtlichen Ordnerstrukturen. Statt zu überlegen, in welchen Unter-Unter-Ordner eine Datei gehört, hängen Sie einfach Schlagworte (Hashtags) an den Dateinamen an. So finden Sie Ihre Dokumente in Sekundenschnelle wieder – egal, wo sie gespeichert sind.

Ein Praxis-Beispiel

Stellen Sie sich vor, Sie bereiten den Unterricht vor. Anstatt die Datei in Schule > Klassen > 5a > Fächer > Deutsch > Planung zu verstecken, nutzen Sie einfach Tags:

Vorher: Lehrplan_2026.docx

Nachher: Lehrplan_2026 #planung #klasse5a #deutsch .docx

Hinweis: Das Programm setzt absichtlich ein Leerzeichen vor die Dateiendung (z.B. .docx). So können Sie den Dateinamen später problemlos von Hand bearbeiten, ohne versehentlich die Dateiendung zu löschen.

1. Einmalige Installation

Keine Sorge: Für diese Einrichtung benötigen Sie keine Administrator-Rechte. Sie verändern keine Systemeinstellungen und können am (Schul-)Computer absolut nichts kaputt machen!
  1. Erstellen Sie auf Ihrem Laufwerk C: den Ordner C:\Scripts\AddHashtag.
  2. Kopieren Sie den untenstehenden Programm-Code (klicken Sie auf "Code kopieren").
  3. Öffnen Sie den Windows-Texteditor (Notepad), fügen Sie den Code ein und speichern Sie die Datei unter dem Namen Taggen.ps1 genau in dem soeben erstellten Ordner ab.
$FileList = @($args)
if ($FileList.Count -eq 0) { exit }

Add-Type -AssemblyName PresentationFramework

$pool = @()
foreach ($path in $FileList) {
    if (Test-Path -LiteralPath $path) {
        $baseName = (Get-Item -LiteralPath $path).BaseName
        $matches = [regex]::Matches($baseName, '(?:^|\s+)#([\w-]+)')
        foreach ($match in $matches) { $pool += $match.Groups[1].Value }
    }
}
$uniqueTags = $pool | Select-Object -Unique

[xml]$xaml = @"
<Window xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        Title="add tag ($($FileList.Count) Dateien)" Height="260" Width="420"
        WindowStartupLocation="CenterScreen" ResizeMode="NoResize"
        Background="#1e1e1e" Foreground="#d4d4d4" FontFamily="Segoe UI">
    <Grid Margin="15">
        <Grid.RowDefinitions><RowDefinition Height="Auto"/><RowDefinition Height="*"/><RowDefinition Height="Auto"/></Grid.RowDefinitions>
        <StackPanel Grid.Row="0" Orientation="Horizontal" Margin="0,0,0,15">
            <TextBlock Text="#" FontSize="20" VerticalAlignment="Center" Margin="0,0,8,0" Foreground="#007acc"/>
            <TextBox Name="TagInput" Width="340" Height="30" FontSize="14" Background="#252526" Foreground="White" BorderBrush="#3f3f46" Padding="5,0"/>
        </StackPanel>
        <ScrollViewer Grid.Row="1" VerticalScrollBarVisibility="Auto"><WrapPanel Name="TagPanel" Orientation="Horizontal"/></ScrollViewer>
        <StackPanel Grid.Row="2" Orientation="Horizontal" HorizontalAlignment="Right" Margin="0,15,0,0">
            <Button Name="BtnCancel" Content="Abbrechen" Width="80" Height="28" Margin="0,0,10,0" Background="#333333" Foreground="White"/>
            <Button Name="BtnOk" Content="OK" Width="80" Height="28" Background="#007acc" Foreground="White" IsDefault="True"/>
        </StackPanel>
    </Grid>
</Window>
"@

$reader = (New-Object System.Xml.XmlNodeReader $xaml)
$window = [System.Windows.Markup.XamlReader]::Load($reader)
$tagInput = $window.FindName("TagInput")
$tagPanel = $window.FindName("TagPanel")

foreach ($tag in $uniqueTags) {
    $btn = New-Object System.Windows.Controls.Button
    $btn.Content = $tag; $btn.Margin = "0,0,8,8"; $btn.Padding = "12,6"; $btn.Background = "#333333"; $btn.Foreground = "#d4d4d4"; $btn.BorderThickness = "0"
    $btn.Add_Click({ $tagInput.Text += ($tagInput.Text ? " " : "") + $tag; $tagInput.Focus() })
    $tagPanel.Children.Add($btn)
}

($window.FindName("BtnOk")).Add_Click({ $window.DialogResult = $true })
($window.FindName("BtnCancel")).Add_Click({ $window.DialogResult = $false })

$tagInput.Focus()
if ($window.ShowDialog() -eq $true) {
    $newTagText = $tagInput.Text.Trim()
    foreach ($path in $FileList) {
        if (Test-Path -LiteralPath $path) {
            $file = Get-Item -LiteralPath $path
            $ext = $file.Extension
            $baseName = ($file.BaseName -replace '((?:^|\s+)#[\w-]+)+$', '').Trim()
            $newName = [string]::IsNullOrWhiteSpace($newTagText) ? "$baseName$ext" : "$baseName $(($newTagText -split '\s+' | ForEach-Object { "#$($_.TrimStart('#'))" }) -join ' ') $ext"
            if ($file.Name -ne $newName) { Rename-Item -LiteralPath $file.FullName -NewName $newName }
        }
    }
}

2. Schaltfläche für Rechtsklick anlegen

Damit Sie das Programm bequem über das Rechtsklick-Menü bei jeder Datei starten können, legen wir eine Verknüpfung an:

  1. Drücken Sie die Windows-Taste + R auf Ihrer Tastatur. Ein kleines Fenster öffnet sich.
  2. Tippen Sie dort shell:sendto ein und drücken Sie Enter. Ein Ordner öffnet sich.
  3. Machen Sie einen Rechtsklick ins Leere dieses Ordners und wählen Sie Neu -> Verknüpfung.
  4. Nun werden Sie nach einem Speicherort gefragt. Markieren und kopieren Sie dafür den untenstehenden Text. Wichtig: Es handelt sich hierbei um eine einzige, lange zusammenhängende Textzeile:
    powershell.exe -ExecutionPolicy Bypass -WindowStyle Hidden -Sta -File "C:\Scripts\AddHashtag\Taggen.ps1"
  5. Klicken Sie auf "Weiter", geben Sie der Verknüpfung den Namen add tag und klicken Sie auf Fertigstellen.

Die Nutzung: Das war's! Markieren Sie ab sofort eine oder mehrere Dateien, machen Sie einen Rechtsklick, navigieren Sie zu "Senden an" und klicken Sie auf "add tag". Das Menü öffnet sich und Sie können Ihre Hashtags vergeben.

3. Die perfekten Begleiter für den Workflow

Hashtags entfalten ihre volle Stärke erst in Kombination mit einer pfeilschnellen Windows-Suche. Der Standard-Explorer ist hier oft zu langsam. Wir empfehlen zwingend diese beiden kostenlosen Tools:

1. Everything (voidtools)

Die schnellste Suchmaschine für Windows. Findet jede Datei in Millisekunden. Wenn Sie hier nach #klasse5a suchen, haben Sie sofort alle relevanten Dokumente im Blick.

Zu Everything wechseln →

2. Everything Toolbar

Dieses Tool integriert die Suchfunktion direkt in Ihre Windows-Taskleiste.

Anleitung für den Download (die Seite ist auf Englisch):
  1. Klicken Sie unten auf den Download-Link.
  2. Scrollen Sie auf der Seite etwas nach unten, bis Sie den Bereich "Assets" sehen.
  3. Suchen Sie in der Liste nach der Datei, die auf x64.exe endet (z.B. EverythingToolbar-3.4.5-x64.exe) und klicken Sie darauf, um sie herunterzuladen.
